# 🚨 VibeCoding 실시간 문제 해결 가이드

## 🎯 기본 원칙
1. **절대 패닉하지 않기** - 아이들이 불안해함
2. **30초 룰** - 30초 안에 해결 안 되면 백업 방법 사용
3. **칭찬 우선** - "지금까지 잘했어!" 먼저 말하기
4. **함께 해결** - "우리 같이 고쳐보자!"

## 🔥 1순위 긴급상황 (즉시 대응)

### 😱 "화면이 까만색이에요!"
**원인:** 자바스크립트 에러로 게임이 멈춤

**30초 해결법:**
1. **F5 새로고침** (가장 빠름)
2. **Live Server 재시작**: 우클릭 → "Stop Live Server" → "Open Live Server"
3. **브라우저 재시작**

**백업 방법:**
```html
<!-- 긴급 템플릿 (메모장에 복사-붙여넣기) -->
<!DOCTYPE html>
<html>
<body style="background: blue; text-align: center; padding: 100px;">
    <h1 style="color: white;">🎮 내 게임</h1>
    <div id="hero" style="width: 50px; height: 50px; background: red; margin: 50px auto; cursor: pointer;" onclick="this.style.background = 'yellow'"></div>
    <p style="color: white;">빨간 네모를 클릭해보세요!</p>
</body>
</html>
```

### 😰 "아무것도 안 움직여요!"
**원인:** 키보드 포커스 문제

**즉시 해결:**
1. **게임 화면 클릭** → "게임 화면을 한 번 클릭해봐!"
2. **키보드 테스트** → 화살표키 눌러보기
3. **마우스로 대체** → 클릭으로 움직이게 코드 변경

**대체 코드:**
```javascript
// 마우스 클릭으로 이동
canvas.addEventListener('click', function(e) {
    const rect = canvas.getBoundingClientRect();
    hero.x = e.clientX - rect.left - 25;
    hero.y = e.clientY - rect.top - 25;
    draw();
});
```

### 🔴 "빨간 글씨가 많이 나와요!"
**원인:** 콘솔 에러 (아이들이 볼 필요 없음)

**즉시 대응:**
1. **F12 개발자 도구 닫기** → "이건 개발자용이야, 닫아도 돼!"
2. **새로고침** → F5
3. **긍정적 전환** → "에러는 개발자의 친구야! 배우는 거야!"

## ⚡ 2순위 일반 문제 (진정하고 해결)

### 💻 "Cursor가 안 열려요!"
**백업 방법 (순서대로):**
1. **메모장/TextEdit 사용**
   - Windows: 시작 → 메모장
   - Mac: Spotlight → TextEdit
2. **코드 복사-붙여넣기**
3. **"다른 이름으로 저장" → 파일명: `game.html`**
4. **파일 더블클릭으로 실행**

### 🌐 "인터넷이 안 돼요!"
**오프라인 모드 전환:**
1. **`basic-game-standalone.html` 사용**
2. **파일을 더블클릭으로 직접 실행**
3. **"인터넷 없이도 게임을 만들 수 있어!"**

### 🖥️ "브라우저가 이상해요!"
**다른 브라우저 시도:**
1. **Edge** (Windows 기본)
2. **Safari** (Mac 기본)  
3. **Firefox** (있다면)
4. **파일 탐색기에서 직접 실행**
