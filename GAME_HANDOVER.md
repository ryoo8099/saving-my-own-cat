## 📋 프로젝트 개요

**게임 제목**: "괴물을 물리치고, 너의 애완동물을 구하라!"  
**개발 기간**: 19단계에 걸친 반복적 개발  
**최종 완성일**: 2024년  
**개발자**: 호우주의님 (클라이언트) + AI 어시스턴트  

---

## 🎯 게임 개요

### 게임 스토리
- 플레이어(고양이)가 몬스터들을 물리치고 철창에 갇힌 애완동물 고양이를 구출하는 게임
- 3단계 스테이지를 거쳐 열쇠를 획득하고 최종 구출까지 진행

### 핵심 기능
- 🐱 고양이 플레이어 (HP 시스템, 색상 변화)
- 🔫 가관총 시스템 (3발씩 발사, 자동 재장전)
- 🚁 폭격기 시스템 (A키로 호출)
- 🎯 3단계 스테이지 진행
- 🔑 열쇠 획득 및 자동 철장 열기
- 🎉 구출 애니메이션 및 축하 시스템

## 🎮 게임 시스템 상세

### 플레이어 시스템
- **캐릭터**: 고양이 (40x40 픽셀)
- **HP**: 1500 (최대)
- **이동**: 방향키 (좌우)
- **점프**: 스페이스바
- **무기**: 가관총 (1번키/2번키로 3발씩 발사)
- **특수능력**: 폭격기 호출 (A키)

### 전투 시스템
- **총알**: 노란색, 8픽셀/프레임 속도
- **장전**: R키 (0.5초)
- **자동재장전**: 총알 0개 시 0.1초 후 자동
- **HP 회복**: 
  - 탱크 처치 시 +50 HP
  - 몬스터 처치 시 +100 HP
  - 보스 처치 시 +150 HP

### 적 시스템
- **몬스터**: 빨간색, HP 100, 데미지 15
- **탱크**: 갈색, HP 50, 데미지 25
- **보스**: 거대/로봇, HP 200/150, 데미지 30/25
- **적 총알**: 2초마다 발사, 플레이어 HP -10

### 스테이지 시스템
- **스테이지 1**: 몬스터 1마리, 보스 1마리
- **스테이지 2**: 몬스터 1마리, 보스 1마리  
- **스테이지 3**: 몬스터 1마리, 보스 1마리
- **완료 조건**: 각 스테이지의 몬스터/보스 모두 처치

### 구출 시스템
- **열쇠 획득**: 모든 스테이지 완료 시
- **자동 구출**: 열쇠 보유 시 철창 근처(700-800px)에서 자동 실행
- **애니메이션**: 3초간 철창에서 화면 중앙으로 이동
- **축하**: 구출 완료 후 화면 중앙에서 만세

## 🔧 기술적 구현

### 게임 상태 관리
```javascript
let gameState = {
    score: 0,
    playerHP: 1500,
    maxHP: 1500,
    stage: 1,
    ammo: 150,
    maxAmmo: 150,
    hasKey: false,
    petRescued: false,
    // ... 기타 상태들
};
```

### 충돌 감지 시스템
- **AABB (Axis-Aligned Bounding Box)** 충돌 감지
- 총알-적, 총알-보스, 플레이어-적, 플레이어-보스
- 적총알-플레이어 충돌 처리

### 애니메이션 시스템
- **구출 애니메이션**: 선형 보간법 사용
- **HP 바**: 실시간 업데이트
- **색상 변화**: HP에 따른 플레이어 색상 변경

## 🎮 게임 실행 방법

1. **파일 확인**: 모든 8개 파일이 같은 폴더에 있는지 확인
2. **브라우저 실행**: `game.html` 파일을 브라우저에서 열기
3. **게임 시작**: 자동으로 게임이 시작됨
4. **조작법 확인**: 화면 하단의 조작법 안내 참고
