## 📁 프로젝트 구조

```
VibeCoding/
├── game.html          # 메인 게임 파일
├── game.css           # 모바일 최적화 스타일
├── game-state.js      # 게임 상태 관리
├── game-objects.js    # 게임 객체들
├── game-input.js      # 터치/키보드 입력
├── game-functions.js  # 게임 로직
├── game-drawing.js    # 그래픽 렌더링
└── game-loop.js       # 게임 루프
```

## 🎮 게임 플레이 방법

### 모바일에서:
1. 위 링크를 클릭하거나 직접 접속
2. 터치 버튼으로 게임 플레이
3. 세로 모드 권장

### 데스크톱에서:
1. 키보드로 플레이 가능
2. 마우스로 터치 버튼 클릭 가능
